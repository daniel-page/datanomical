# Datanomical  
A datalogging application that collects and visualises data sent via REST API or WebSocket API.  

Development is currently in progress.  

## Dependencies  
- Node.js v20.5.1  
- Chart.js v4.3.3  
- Yarn v1.22.18  
- chartjs-adapter-date-fns v3.0.0
